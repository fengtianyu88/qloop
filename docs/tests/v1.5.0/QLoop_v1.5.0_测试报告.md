## 六、发现并修复的 Bug

### 6.1 产品 Bug (5 个)

| Bug ID | 严重度 | 描述 | 修复方案 | 验证用例 |
|--------|--------|------|---------|---------|
| BUG-01 | **高** | MinIO presigned URL 有效期为 1 小时,违反 7 天约束 | `expiry_hours=1` → `expiry_hours=168` (releases.py L434/L505) | TC-DL-06 |
| BUG-02 | **中** | 上传空文件被接受,无空文件检查 | 添加 `if not file_data: raise ValueError` (release_service.py 3 处) | TC-UPLOAD-14 |
| BUG-03 | **高** | confirm_release 状态冲突返回 500 | 添加状态预检,返回 409 Conflict (v1.4.7.2) | TC-STATE-14 |
| BUG-04 | **高** | 文件类型白名单校验失败返回 500 | 捕获 ValueError 返回 415 (v1.4.7.2) | TC-UPLOAD-05 |
| BUG-05 | **高** | API 层并发触发评审无预检 | 添加 PENDING 预检返回 409 + SSE 优先查 PENDING (v1.4.7.3) | TC-LLM-06 |

### 6.2 测试脚本适配问题 (8 个)

| 问题 | 修复 |
|------|------|
| WSL 环境检测失败 | 添加 `_detect_wsl_prefix()` 自动检测 |
| LLM 配置端点路径错误 | `/api/llm-models` → `/api/llm-config/models` |
| 上传文件通过 WSL base64 失败 | 改用 Python `io.BytesIO` + `zipfile` |
| 路径含空格转义问题 | 双引号包裹路径 |
| grep 正则 `\|` 不支持 | 改用 `grep -lE` 扩展正则 |
| Vite 构建后变量名压缩 | 添加源码后备搜索 |
| MINIO_SECRET_KEY 子串误判 | 改为逐行精确匹配 |
| DEBUG 未设置被标记 SKIP | 默认为 False(安全) |
